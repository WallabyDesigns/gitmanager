Configuration

Git Project Manager

Environment variables and optional runtime overrides.

Installation Projects

Core Settings

  • APP_URL, APP_KEY, and database credentials.
  • GITHUB_TOKEN for private repos and Dependabot.
  • GITHUB_WEBHOOK_SECRET for GitHub webhook verification.

Binary Overrides

Point GPM at specific binaries when multiple installations exist:

  • GPM_GIT_BINARY
  • GPM_COMPOSER_BINARY
  • GPM_NPM_BINARY
  • GPM_PHP_BINARY or GPM_PHP_PATH
  • GPM_PROCESS_PATH to prepend PATH.

Preview Builds

  • GPM_PREVIEW_PATH (local base path)
  • GPM_PREVIEW_BASE_URL (optional URL prefix)

Self Update

  • GPM_SELF_UPDATE_ENABLED to toggle nightly updates.
  • GPM_ASKPASS_DIR for Git credential helper location.
