Queue Behavior

When the deploy queue is enabled, deploy requests are queued and processed sequentially.

  • Manual deploys, auto-deploys, and webhooks all enqueue
  • Only one deploy runs at a time per project
  • Queued items can be cancelled or reordered

Enable / Disable

Toggle the queue via GWM_DEPLOY_QUEUE_ENABLED in .env.

Queue Processor

The scheduler runs deployments:process-queue every minute to process queued items.
