First User Registration

Registration is open only when no users exist. After the first user is created, registration is automatically disabled.

Admin-Created Users

  • Create users from the Users page in the main navigation.
  • Optional temporary password generation.
  • Users are required to change password on first login.

Password Reset

Admins can issue a reset link or generate a new temporary password when needed.
